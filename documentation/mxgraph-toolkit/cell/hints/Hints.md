# Hints

## create

```ts
const hints = new Hints(hintsElement)
```

## init

Initializes hints by setting opacity (default: 50)

```ts
hints.init({opacity: 60})
```

## add hint

Add text to hint

```ts
hints.addText(text)
```


## append hint to document

Appends hint element to document

```ts
hints.appendToDocument()
```
