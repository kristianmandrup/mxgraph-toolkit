# SwimlaneLayout

## create

```ts
const swimlaneLayout = new SwimlaneLayout()
```

## properties

Returns graph model

```ts
swimlaneLayout.model
```

## init

Initializes layout

```ts
swimlanes.init()
```

## set manager

```ts
swimlanes.setManager(layoutManager)
```

## is vertex ignored?

Determine if vertex should be ignored for this swimlane (ie. not drawn)

```ts
swimlanes.isVertexIgnored(vertex)
```
