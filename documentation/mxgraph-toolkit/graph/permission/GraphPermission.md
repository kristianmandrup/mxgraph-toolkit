# GraphPermission

## create

```ts
const graphPermission = new GraphPermission(graph: any, {locked, createEdges, editEdges, editVertices, cloneCells})
```

## init

Initialize graph methods

- `isCellDisconnectable`
- `isTerminalPointMovable`


```ts
graphPermission.init()
```
