# Edge

## create

```ts
const edge = new Edge(graph)
```

Sub APIs

- connection
- markers
- guides

## connection API

### get connection API

```ts
edge.connection
```

### set connection API

```ts
edge.setConnection(connection)
```

## markers API

### get markers API

```ts
edge.markers
```

## set markers

```ts
edge.setMarkers(markers)
```

## animate

### get animate API

```ts
edge.animate
```

## set animate

```ts
edge.setAnimate(animate)
```

## guides

### get guides API

```ts
edge.guides
```

## set guides

```ts
edge.setGuides(guides)
```
